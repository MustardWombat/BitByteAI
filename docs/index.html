Restore Purchases

In the app: Profile → Settings → Restore Purchases. Make sure you’re signed into the same Apple ID.

Manage or Cancel Subscription

Open: Apple Subscriptions → select BitByte Pro.

Troubleshooting

  1. Force-quit and reopen the app.
  2. Check connectivity and App Store sign-in.
  3. Use Restore Purchases (see above).
  4. Reboot your device. Still stuck? Email or open a GitHub issue with steps to reproduce.
